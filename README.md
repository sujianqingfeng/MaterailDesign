# MaterailDesign
这个项目名取得不太好，当时用来做MD设计练习的，结果不知不觉接触到其他的知识，顺带就练习了。名字就将就吧。
mvp的实现采用了一个框架TheMvp，详情http://www.kymjs.com/code/2015/11/09/01/。

# 项目特点
 - mvp
 - rxjava
 - retrofit2
 - MaterailDesign
 - Glide
 - 夜间模式（夜间模式还有一个小bug）

# 项目功能
> 图片和音乐使用的是百度API ，视频和新闻用的是易源的API。

  - 图片
  - 音乐
  - 视频
  - 新闻
